Teacher Notes

  • Each segment will be built to feel like an OST-style testing session rather than a simple quiz.
  • Question types will include standard MCQ, passage sets, Part A/Part B, multi-select, dropdown-style, and drag/drop-style interactions.
  • Apps Script tracking is designed to show misses by exact question, Content Statement, question type, and segment.
  • Use previous-day results to intensify the next day’s coverage where students need it most.
